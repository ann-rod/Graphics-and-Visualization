In order to run this program, open the group_10_assignment6.pde file
in Processing and click Play.

To add animals to the simulation, press the respective key:
  - "h" to spawn random hawk
  - "r" to spawn random bird
  - "b" to spawn random bee

Make sure all necessary classes are in folder:
  - Bird.pde
  - Flock.pde
  - Ecosystem.pde
  - Hawk.pde
